Real-Time Executive on ARM Cortex M4

A Real-Time Executive (RTX) on the ARM Cortex M4 microprocessor, supporting priority scheduling (via deadlines), pre-emption through context switching, and dynamic memory management. The project includes writing fundamental kernel API functions like osKernelInit, osCreateTask, implementing an Earliest Deadline First (EDF) scheduler, and developing dynamic memory functions (malloc and free) for memory management. The project was developed in C and ARM Assembly, using STM32Cube IDE with STM32F401RE as the microcontroller.

Car Detection System

An AI-powered vehicle detection system designed to detect and track cars in real-time. Built using Python's OpenCV library, this software utilizes advanced computer vision techniques such as object detection and tracking to analyze video streams and accurately identify vehicles. The system can process live video feeds and detect cars, making it ideal for applications in traffic monitoring, smart city solutions, and security. This project won 1st place in the **Toyota Innovation Challenge 2022**, showcasing its potential to revolutionize urban transportation systems and traffic management.

MP3 Player

A MP3 player system using an FPGA to decode and play MP3 audio files stored on an SD card. The system was programmed in C and implemented on an FPGA platform. Technologies Used: C programming, FPGA, SD card integration, MP3 Audio Decoding, Hardware Control Logic, Low-latency Audio Playback

GPS Simulator

A GPS simulator system that finds the shortest path between two locations. Given a map of N houses and their connections, the program organizes the data into a graph implemented using an adjacency list. It uses Prim's algorithm and a heap data structure to find the Minimum Spanning Tree (MST), allowing the GPS to suggest the shortest route from house A to house B.
